WALLET FINANCE MANAGER

  • Github

Personalized money managing and financial tracking cross-platform application that encourages better spending habits and budget tracking. It brings together user's bank accounts, credit cards, bills and investments so user can see all the spending, set and manage the budget, and reach financial goals. Plaid API was used in order to connect user's bank accounts - it provides a secure way to store user's transactional data through a multi-factor authentication process. Wallet also provides a loan calculator which calculates the monthly payment, the amount of interest paid, the total cost of credit, and payment schedules.

  • React-Native
  • Redux
  • Node
  • Express
  • Sequelize ORM
  • PostgreSQL
  • Plaid API
  • ChartsJS

MASTERMIND AI

  • Github

Mastermind Game implementation in Java Swing including an AI, which can be played by a user "against" the computer and vice versa. It utilizes Google Cloud Text-to-Speech API which allow blind or visually impaired users to navigate and understand the current state of the game. This is a game where player tries to guess the number combination. After each guess the computer provides feedback whether the player had guess a number correctly, or/and a number and its position correctly. Also, the player can choose the option to be the Codemaker and watch the AI breaking the code. Simple AI was implemented using Genetic Algorithm which generates multiple populations of candidate solutions and grades their fitness through mutation, crossover and selection.

THE STACK

  • Java
  • Google Text-to-Speech API
  • Random Number API
  • HTML
  • Swing GUI

SNAP COOK

  • Github

React-Native cross-platform mobile application that can search for recipes based on ingredients detected in food photos (either from your camera roll or directly from your camera). It also provides vegetarian, vegan, sugar-conscious and peanut-free recipes filtering options. Users can save and share their favourite recipes. Created using Clarifai's image recognition API (the Food model) and Edamam's recipe search API that provides the most accurate automated nutrition analysis.

THE STACK

  • Javascript
  • React-Native
  • Redux
  • Node
  • Edamam's API
  • Clarifai's API
  • HTML
  • CSS
